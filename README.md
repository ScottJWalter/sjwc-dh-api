# Usage
<!-- usage -->
```sh-session
$ npm install -g sjwc-dh-api
$ dh-api COMMAND
running command...
$ dh-api (-v|--version|version)
sjwc-dh-api/0.0.1 linux-x64 node-v12.16.3
$ dh-api --help [COMMAND]
USAGE
  $ dh-api COMMAND
...
```
